# מיפוי כלים לתסמינים - מערכת אבטחה לנפש (Security System for the Soul)

סוויטת המוצרים "מערכת אבטחה לנפש" מציעה כלים טכנולוגיים המותאמים בדיוק לתסמינים הקליניים של ההפרעות השונות.

## 1. הפרעה דו-קוטבית (מאניה דיפרסיה)
**מאפיינים:** תנודות קיצוניות בין אנרגיה גבוהה (מאניה) לדיכאון עמוק.

*   **Financial Shield (מגן פיננסי):**
    *   *מענה ל:* **מאניה** (קבלת החלטות פזיזות, הוצאות כספיות מוגזמות).
    *   *מנגנון:* נועלת את ה"ארנק" בזיהוי דיבור מהיר ואופורי.
*   **Medication Verifier (vDOT):**
    *   *מענה ל:* **אי-יציבות / הישנות התקפים**.
    *   *חשיבות:* הטיפול המרכזי הוא תרופות מייצבות מצב רוח. מניעת התקפים פסיכוטיים ואשפוזים.
*   **Daily Meaning Compass (הנסיך הקטן):**
    *   *מענה ל:* **דיכאון** (חוסר תקווה, עייפות, תחושת "מוות בתוך גוף חי").
    *   *מנגנון:* מייצר סדר לוגי ומשימות למניעת הריקנות.

## 2. הפרעת אישיות גבולית (BPD)
**מאפיינים:** חוסר יציבות בזהות, ביחסים ובמצבי הרוח, פחד מנטישה.

*   **Compassion Mirror (מראת החמלה):**
    *   *מענה ל:* **פגיעה עצמית ואובדנות**.
    *   *מנגנון:* בולם אלימות עצמית באמצעות "שוק" של רגש חיובי (תמונת הילד הפנימי).
*   **Emuna AI (מסנן רעשים):**
    *   *מענה ל:* **תחושת ריקנות כרונית** וביקורת עצמית (במיוחד ב"סוג השקט").
*   **De-escalation (נטרול פצצה):**
    *   *מענה ל:* **התפרצויות כעס** ("הסוג האימפולסיבי").
    *   *מנגנון:* השתלטות חושית להורדת "להבות" הרגש.
*   **Black Box (קופסה שחורה):**
    *   *מענה ל:* **דיסוציאציה** (ניתוק מהמציאות) או אובדן שליטה.
    *   *מנגנון:* תיעוד ולמידה על האירוע בדיעבד.

## 3. סכיזופרניה והפרעות פסיכוטיות
**מאפיינים:** ניתוק מהמציאות, הזיות, מחשבות שווא.

*   **Emuna AI:**
    *   *מענה ל:* **קולות פנימיים שליליים / הזיות**.
    *   *מנגנון:* החלפתם בקול חיצוני מכיל ומרגיע ("מסנן רעשים").
*   **Medication Verifier:**
    *   *מענה ל:* **התקפים פסיכוטיים**.
    *   *חשיבות:* מניעת אשפוזים כפויים/קשירות ע"י הקפדה על תרופות.
*   **Daily Meaning Compass:**
    *   *מענה ל:* **חשיבה לא מאורגנת** וקושי בתפיסת המציאות.
    *   *מנגנון:* "סדר בתוך הכאוס".

## 4. אוטיזם ותסמונת דאון
**מאפיינים:** אתגרי תקשורת, חשיבות השילוב בקהילה ובחיים הדתיים.

*   **Daily Meaning Compass:**
    *   *מענה ל:* צורך ב**סדר יום מובנה** וקיום מצוות (לבעלי "דעת").
*   **Emuna AI:**
    *   *מענה ל:* צורך ב**תיווך המציאות**.
    *   *גישה:* "נשמות גבוהות" הזקוקות לתמיכה ללא שיפוט.
